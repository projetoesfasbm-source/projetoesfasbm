🛡️ SafeExamBrowser (SafeBrowser)

Para a realização das avaliações no sistema, é obrigatório o uso do SafeExamBrowser. Siga os passos abaixo para instalar e configurar seu ambiente.

1
Baixar o SafeBrowser

Faça o download e instale o SafeExamBrowser oficial para o seu sistema operacional (Windows, macOS ou iOS).

🔗 CLIQUE AQUI PARA BAIXAR O SAFEBROWSER (Download)
2
Baixar Arquivo Teste

Este arquivo irá testar se o SafeBrowser esta funcionando. Após Finalizar a parte 1 baixe e execute.

{# IMPORTANTE: Certifique-se de que o arquivo "configuracao_prova.seb" esteja na pasta "static/seb/" do seu projeto. #} ⬇️ Baixar Arquivo (.seb)
3
Como Acessar

  1. Instale o SafeExamBrowser (Passo 1).
  2. Localize o arquivo .seb baixado (Passo 2).
  3. Dê um clique duplo no arquivo .seb.
  4. O navegador abrirá automaticamente na página teste.
    5. 🚨Para sair do SafeBrowser Digite CTRL+Q🚨

ℹ️
Dica: Ao Executar o arquivo teste poderá Surgir a Mensagem "Configuration Sucessful", Clique em NÃO.
